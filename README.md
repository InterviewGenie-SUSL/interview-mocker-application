# InterviewGenie – AI Mock Interview Platform

## 🚀 Overview
InterviewGenie is a full-stack AI-powered web application that helps users practice interviews and receive real-time feedback using AI.

## 🛠 Tech Stack
- Frontend: Next.js, React, Tailwind CSS
- Backend: Node.js, Express.js
- Database: PostgreSQL
- AI: Gemini AI
- Authentication: Clerk

## ✨ Features
- AI-generated interview questions
- Real-time answer recording
- AI-based feedback system
- User authentication & session tracking
- Responsive modern UI
